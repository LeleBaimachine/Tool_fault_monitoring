This dataset contains information on sudden tool faults during turning and milling processes, which illustrate brittle fracture processes. 
The data includes force and acceleration measurements collected from the machine, with milling data also supplemented by microphone recordings. 
Data_turning, first: force (N); second: vibration (g). fs = 12800 Hz. Data_milling, first: force (N); second: vibration (g); third: sound (Pa). fs = 6400 Hz.
